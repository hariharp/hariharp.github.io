CSS Concepts Used

Flexbox

The blog posts are arranged using Flexbox, which allows for a flexible layout that adjusts based on the screen size. The .blog-posts class defines a flex container that organizes blog cards in a responsive manner.

Box Model

The CSS Box Model is applied to each blog card with margins, borders, padding, and content areas. Each .blog-card has padding for internal spacing and borders to separate it visually from others.

Responsive Design

Media queries and responsive properties ensure that the layout adapts to different screen sizes, making it accessible on both mobile and desktop devices.
