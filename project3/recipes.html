CSS Concepts Used

CSS Grid

The recipe cards are arranged using CSS Grid, which allows for a flexible layout that adjusts based on the screen size. The .recipe-grid class defines a grid layout that creates responsive columns for the recipe cards.

Box Model

The CSS Box Model is applied to each recipe card with margins, borders, padding, and content areas. Each .recipe-card has padding for internal spacing and borders to separate it visually from others.

Responsive Design

Media queries and responsive properties ensure that the layout adapts to different screen sizes, making it accessible on both mobile and desktop devices.
